### 3. Amazon Sales Exploratory Data Analysis

Exploratory data analysis project developed in Python to clean, transform, analyze, and visualize Amazon sales data.

**Key features:**
- Data loading and initial inspection.
- Data cleaning and type conversion.
- Price, discount, rating, and review count transformation.
- Missing value treatment.
- Duplicate validation.
- Correlation analysis using Pearson and Spearman methods.
- Data visualization with scatter plots and heatmaps.
- Statistical testing using t-tests and chi-square analysis.
- Label encoding for categorical variables.

**Technologies used:**
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Scikit-learn
